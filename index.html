ProjectSQL

Let's start with a project where I analyze some hurricane data, clean and restructure it using SQL, analyze it using Python and then present some data in a dashboard using Power BI. The point of this project is to investigate how hurricane frequency has changed over time.

We start with data taken from Kaggle. I use the import wizard to import it into a MySQL database using MySQL Workbench as seen in the image below.

Since we are interested only in frequency we clean the data to only retain the data we want to use. Inspecting the data reveals that a single hurricane has a unique ID but it is given several rows representing the same hurricane at different times. We only want one row per hurricane and we are interested in the start time- To do this we should sort by date and then remove every row, except the first, with a specific ID

Let's get going in Python and try to predict how many hurricanes we have each year.

Well, that took a turn. Anyway, let's get into PowerBi and vizualize this stuff.
